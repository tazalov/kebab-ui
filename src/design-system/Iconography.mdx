# Iconography

<br />

## Usage

All the icons in the design system are available as React components. You can use them like this:

<Markdown>{IconsUsage}</Markdown>

## Properties

All available attributes for the `<svg>` element are available as attributes for icons, with the exception of the viewBox.

To change the color, you can use the css `color` property, as you would for text (for all icons, the `fill` property has the `currentColor` value)

For example, to make an icon red, you need to use `inline-styles` or the `className` attribute:
<Markdown>{IconsColor}</Markdown>
